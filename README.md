# Calculator API

Esta es una API RESTful para una calculadora que implementa Notación Polaca Inversa (RPN) utilizando ASP.NET Core. Permite realizar operaciones matemáticas en notación polaca inversa, como suma, resta, multiplicación, división y potencia. Además, guarda un historial de operaciones en una base de datos SQLite utilizando Entity Framework Core.

## Requisitos

- .NET 8.0 o superior
- SQLite
- Entity Framework Core
- Swagger/OpenAPI para la documentación de la API
